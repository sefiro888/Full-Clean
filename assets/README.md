## Recomendaciones
- Formato `.jpg` para fotos (o `.webp` aún más ligero).
- Pásalas por https://squoosh.app o https://tinyjpg.com para reducir peso un 60–70%.
- Nombres en minúsculas, sin tildes ni espacios.
- Ancho máximo 1600 px (no subas fotos directas del móvil de 4-8 MB).
